# Cell2clone

## Installation

For now, the package needs to be installed from GitHub

```bash
git clone https://github.com/Teichlab/cell2clone
cd cell2home
pip install .
```

## Usage and Documentation

Please refer to [ReadTheDocs](https://cell2clone.readthedocs.io/en/latest/), which comes with a demo notebook and function docstrings.
